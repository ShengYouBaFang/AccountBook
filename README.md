# Account Book -Kotlin 开发设计文档
## 一、项目概述
Account Book是一款基于 Kotlin 开发的记账app，针对大学生记账难以坚持的痛点，拟开发一款集快速记账、数据可视化、预算管理和OCR识别于一体的个人财务管理应用，采用现代 Android 开发架构和最佳实践。
### 核心特性
- **极速记账**：快速完成记账工作，设置金额、所属分类或自定义分类、备注、时间，提交后即可自动记录并纳入数据统计模块。
- **可视化分析**：直观的图表展示：饼图、柱形图、环形图等。
- **智能预算**：可选择对金额所属分类设置预算，也可设置月度总支出预算，并使用环形图展示预算使用进度。
- **OCR识别**：调用第三方api实现小票拍照图片自动提取信息进行记账。
- **数据安全**：本地存储+导出功能（excel)
### 质量要求
- 基础功能完整，逻辑通顺，无明显闪退（Crash）。
- 工作量饱满（Activity/Fragment数量不少于5个，涉及SQLite/Room数据库操作，涉及网络请求）。
- 代码规范（命名规范、注释清晰）。
- 适当使用高级特性（如Jetpack组件、MVVM架构、自定义View、NDK等）酌情加分。
- UI/UX设计美观，交互流畅（非原生丑陋界面）。
- 重要：UI定制：应用必须包含一套独特的主题色和LOGO。
### 整体思路
1. 在登陆页面底部展示开发者信息，登陆页面主体为用户登陆，输入手机号和密码即可登陆，新手机号自动注册为新账号。
2. 登陆后进入主界面，主界面主体为记账本
  - 顶部有按钮可以选择显示账本类型，按钮默认显示“全部”（点击按钮可以弹出浮窗界面选择显示不同类型的记账数据，例如支出部分：餐饮、交通、教育、购物、其他或有自定义的类型；入账部分：工资、奖金、其他或有自定义的类型），选择类型后就只显示当前类型的数据。
  - 顶部同时有月份选择按钮，按钮默认显示当前月份，点击后弹出浮窗选择显示具体月份的账本数据，在右侧同步展示所选月份的总支出金额和总入账金额。
  - 主界面主体使用列表展示最新的记账数据情况，以天为单位展示，在列表上方显示日期、当天总支出、当天总收入，下方显示这天的具体的记账情况金额类型（餐饮、交通等），具体时间，如有备注也同步显示，最重要的是显示金额，用“+”“-”表示入账和支出。列表可下滑查看更多天数的数据。
  - 重要：在主界面有悬浮按钮，点击可进入记账界面。
3. 记账界面
  - 有按钮选择标记为支出或入账。
  - 有按钮选择记账日期，利用弹出浮窗
  - 可输入金额数
  - 有区域可以选择记账类型：例如支出部分：餐饮、交通、教育、购物、其他或用户进行自定义类型；入账部分：工资、奖金、其他或用户进行自定义类型。
  - 可以选择添加备注
  - 在完成支出、入账标记，完成输入金额，完成记账类型选择后确认按钮才可以生效，点击确认添加数据，数据同步到主界面和数据统计模块。
  - **重要**：有拍照或上传图像按钮，有图像数据（购物小票）后可调用第三方api进行OCR图像识别，自动提取数据后自动添加记账记录，OCR记录默认为支出。
4. 统计界面
  - 首先选择查看月份，选择支出或入账，选择支出就展示支出数据的图表，选择入账就展示入账数据的图表。
  - 按月份统计，绘制环形图展示支出构成（例如餐饮、交通等所占比例）或入账构成。
  - 按月份统计，用柱形图展示每种记账类型总的金额数。
  - 绘制柱形图，展示当前月份每日的支出或入账总金额数。
  - 用列表展示支出或入账金额排行榜
5. 预算管理页面
  - 可对支出金额所属分类设置预算，也可设置月度总支出预算，并使用环形图展示预算使用进度。
6. 我的页面
  - 展示用户相关信息
  - 可进行导出数据统计的excel表格操作
  - 展示app相关信息，开发者信息。
